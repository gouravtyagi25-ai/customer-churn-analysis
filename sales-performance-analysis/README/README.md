### Sales Analysis

### Project Overview

This project focuses on analyzing sales transaction data to evaluate overall business performance, identify top-performing regions and products, and understand revenue trends over time. The analysis combines SQL for data validation and aggregation with Power BI for interactive dashboarding.

### Business Objectives

- Track overall revenue performance
- Identify top-performing regions and products
- Analyze monthly and yearly revenue trends
- Support business decision-making using data-driven insights

### Tools Used
- MySQL
- SQL
- Power BI
- Excel / CSV


## Key Analyses
# Product Performance

- Monitor is the highest revenue-generating product
- Laptop is the second-largest contributor
- Tablet and Accessories show nearly equal contribution
- Smartphone contributes the least revenue, indicating potential optimization opportunities


# Regional Performance

- South region leads in revenue generation
- East and West regions perform closely
- North region shows comparatively lower revenue

# Revenue Trend

Revenue trends were analyzed across time to identify seasonal and performance patterns
